## What is Argentista?

Argentista is a web application that provides real-time economic data about Argentina. It aggregates information from trusted public APIs to give users an up-to-date picture of the Argentine economy — from monthly inflation rates going back decades to live dollar exchange rates across all major market types.

<Info>
All data is sourced from [ArgentinaDatos](https://argentinadatos.com) and [DolarAPI](https://dolarapi.com) — two reliable, open Argentine economic data providers.
</Info>

## Data sources

| Source | Endpoint | Used for |
|--------|----------|---------|
| ArgentinaDatos | `/v1/finanzas/indices/inflacion` | Monthly inflation indices (CPI) |
| ArgentinaDatos | `/v1/cotizaciones/dolares/{casa}` | Historical dollar prices by type |
| DolarAPI | `/v1/dolares` | Live dollar buy/sell prices |

<Tip>
Data is fetched client-side in real time — every page load retrieves fresh data from the source APIs. No caching layer is applied by default.
</Tip>
